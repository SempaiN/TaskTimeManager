## Ejecución de la aplicación

La aplicación está empaquetada en un archivo JAR. Por lo que es necesario tener java instalado en el equipo.
Dentro de la aplicación se utiliza el comando taskkill de Windows, debido a esto hay que ejecutarlo en un SO Windows

---

# Descargar el jar 

El archivo ejecutable se encuentra en la carpeta madre dentro del proyecto o en la sección releases del repositorio


# Ejecutar por teminal

Una vez tengas el ejecutable (JAR) en tu equipo tienes que ir con el terminal a la carpeta donde lo tengas alojado.

Ej: cd C:\Users\Usuario\Descargas\TaskTimeManager

Una vez estés en la ruta correcta tienes que ejecutar el comando 

java -jar TaskTimeManager.jar 


Si todo es correcto, la aplicación se ejecutará en consola y podrás empezar a utilizarla
